# Удаление и переустановка

Чтобы удалить стаки СМП, нужно знать что у нас есть две директории:

- %appdata%/.staki
- %appdata%/Staki Launcher

Чтобы открыть AppData есть два варианта:

1. Нажми Win + R, напиши %appdata%, нажми Enter
2. Открой Проводник, в строке сверху напиши %appdata%, нажми Enter

Порядок переустановки следующий:

1. Закрой майнкрафт и лаунчер, если открыто
2. Удали нужную тебе папку, с игрой и/или лаунчером
3. Открой лаунчер снова и запусти игру

## .staki

Если нужно удалить или переустановить игру, рекомендую просто удалить целиком эту папку.

Здесь содержатся файлы игры, конфиги и т.п.

## Staki Launcher 

Это папка в которой содержатся файлы лаунчера. Если хочется переустановить лаунчер, нужно удалить эту папку тоже.
